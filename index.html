Final Fantasy 8

Welcome to Brofar's collection of information, resources, and tools for Final Fantasy VIII PC speedrunning. The information here has been sourced from many people and websites (including my own research) as seen in the credits section!

Projects

Name Description Status Link
Esthar Practice Quick memory writer for Esthar encounter stuff. Working Download
Fish Fin Manip Table lookup for the fish fin manipulation. Working Open
Kaivel's Fish Fin Manip Table lookup for Kaivel's alternate fish fin manipulation. Working Open
Encounter Calculator Using the previous encounter formation ID along with the encounter counter (total number of encounters) we can tell which encounter formation we will receive next. Working Open
Cheat Engine Tables (PC) A file for use with Cheat Engine to manipulate various aspects of the game's memory. Working Download
Caraway Code Reads a specific memory address from FF8_EN or FF8_FR (Steam 2013 / non-remaster) containing the code to get into Caraway's mansion. Note: This tool is illegal in speedruns as it reads directly from the game's memory. Working Open
Controls Randomizer Randomizes controller inputs while playing FF8 by reading and shuffling the control mapping in the game's memory while you play. Has some configuration options. Working Open
Card RNG Manip Tool A webpage to keep track of early game RNG events to help with Zell card. Under Construction Open
Final Party Manipulation Conversion of Pingval's final party manipulation code for Playstation and awesomeWaves' modifications for PC ruby script to JS, so that it can be run on the web without requiring runners to install Ruby. BETA Open
Caraway Code - Poles Alternate website for the pole manip BETA Open
Slot Manipulation The End manip created by Kaivel & romaindurand External Open
